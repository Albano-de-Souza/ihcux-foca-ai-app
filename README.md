## 💡 Conceito de Design: Calm Technology

O design do FocaAí foi criado com base nos princípios de *Calm Technology* (Tecnologia Calma), com o objetivo principal de ajudar estudantes e profissionais a gerenciar o tempo sem gerar ansiedade ou vício. As principais escolhas de design para alcançar isso foram:

*   **Uso Extensivo de Espaço em Branco (Negative Space):** A interface é minimalista, evitando o "congestionamento" visual. O espaço em branco é usado estrategicamente para guiar a atenção para as informações essenciais (como o temporizador) e passar uma sensação de calma e ordem.
*   **Hierarquia de Informação:** O tempo restante de foco é sempre o elemento central e de maior destaque na tela, garantindo que o usuário saiba exatamente onde o foco deve estar.
*   **Feedback Visual Suave:** As microinterações são sutis. Por exemplo, o fundo do dashboard pulsa levemente no ritmo de uma respiração tranquila, fornecendo um feedback de que o aplicativo está funcionando sem ser intrusivo.
*   **Foco no "Tempo Ganho" (Conquistas Positivas):** Em vez de mostrar "tempo de tela" (o que pode gerar culpa), o aplicativo foca no "Tempo Ganho" para a vida real e na quantidade de árvores virtuais plantadas, incentivando e premiando o progresso de forma positiva.
*   **Interações Não Punitivas:** A "Tela de Tentativa de Saída" apresenta uma mensagem calma e respirável, focando no incentivo para continuar ("Sua árvore vai secar"), em vez de mensagens agressivas de erro.

## 🔄 Fluxo de Usuário: Iniciar Sessão de Foco em Grupo

O fluxo para criar e iniciar uma sessão de foco com amigos foi desenhado para ser intuitivo e rápido:

1.  **Configuração de Bloqueio:** O usuário acessa a tela de "Bloqueio Seletivo" e usa os interruptores simples para escolher quais aplicativos estarão trancados.
2.  **Configuração de Ambiente:** Na tela de "Configurar Ambiente", o usuário define o tempo de foco usando o controle deslizante e escolhe o som relaxante (Chuva, Café, Floresta).
3.  **Criar/Entrar na Sala:** O usuário navega para a tela de "Ranking do Grupo" e clica no botão para "Criar Nova Sala" (ou "Entrar com Código").
4.  **Chamar Amigos:** Na sala criada, um código ou link é gerado para que o usuário possa compartilhá-lo com amigos por outros meios (ex: WhatsApp).
5.  **Iniciar:** Quando todos os amigos estiverem na sala, o líder clica em "Iniciar Sessão". O aplicativo inicia o temporizador e a fase de bloqueio para todos os participantes simultaneamente.

## 🛠️ Escolha de Componentes

Nossa escolha de componentes foi baseada na redução da carga cognitiva e na facilidade de uso:

*   **Sliders (Controles Deslizantes):** Utilizamos sliders para a seleção do tempo de foco. Eles permitem que o usuário "arraste e solte" para o valor desejado, o que é mais rápido e intuitivo do que a digitação numérica e evita erros de formatação.
*   **Toggles (Interruptores ON/OFF):** Para o bloqueio seletivo de apps, usamos toggles. Esse componente fornece uma indicação clara e imediata do estado (ligado/desligado) e requer um único toque, eliminando a necessidade de confirmar as escolhas individualmente.
*   **Radio Buttons (Botões de Rádio):** Para a seleção de sons ambientais, os radio buttons são ideais, pois deixam claro que apenas uma opção pode ser escolhida de cada vez, evitando confusão.
